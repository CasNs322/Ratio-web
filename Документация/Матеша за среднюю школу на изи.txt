Квадратный корень: короче, чтобы найти квадратный корень из сложного числа, нужно целым числом записать результат из корня числа, которое есть в таблице квадратов.
При этом это число должно быть крайним меньшим по сравнению с числом, из которого нужно найти корень,
а в дробную часть написать число, в столбце которого находится результат квадрата "правильного" числа, в котором будет число, из которого нужно найти корень, перед котором стоят две цифры, которые неважны.
